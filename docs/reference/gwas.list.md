# GWAS files

The genome-wide association studies (GWAS) files of the 10
sex-stratified continental Africa cohorts.

## Usage

``` r
gwas.list
```

## Format

A list of 10 sex-stratified GWAS files. Each component of the list
refers to one GWAS file. Each GWA study file has mandatory column
headers:

- MARKERNAME:

  snp name

- EA:

  effect allele

- NEA:

  non effect allele

- EAF:

  effect allele frequency

- CHROMOSOME:

  chromosome of marker

- POSITION:

  position of marker

- BETA:

  allelic effect

- SE:

  standard error
