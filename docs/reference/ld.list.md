# The linkage disequilibrium (LD) matrices

The LD correlation matrices from 5 continental African cohorts without
sex-stratification.

## Usage

``` r
ld.list
```

## Format

A list containing 5 LD correlation matrices. Each component of the list
refers to one LD matrix. The name of each component refers to one Afican
cohort. The five cohorts are ESN, GWD, LWK, MSL and YRI.
