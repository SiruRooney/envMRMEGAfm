# The smoking proportions across the 10 sex stratified populations.

The smoking proportions across the 10 sex stratified populations.

## Usage

``` r
sexsmk
```

## Format

A data frame containing smoking proportions and sex indicators across 10
cohorts.

- sex:

  sex indicates female and male (female=1, male=0).

- smoke:

  smoking proportions ranging from 20% to 90%.
