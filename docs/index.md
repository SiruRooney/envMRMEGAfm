# envMRMEGAfm

This is an R package for environment-adjusted MR-MEGA fine-mapping
approach.

Website available: <https://SiruRooney.github.io/envMRMEGAfm/>

For more details, please see:

## System requirements

env.MRmega could be installed with ease on versions of R \> 4.3.0.

## Installation

Before installing **envMRMEGAfm** R package , the following packages
from CRAN are required:

``` r
install.packages("data.table")
install.packages("doParallel")
install.packages("dplyr")
install.packages("foreach")
install.packages("parallel")
install.packages("stats")
```

You can install the GitHub version of **envMRMEGAfm** and the
**envMRMEGAfm** R package installation can be finished in 1 minute:

``` r
# install.packages("devtools")
devtools::install_github("SiruRooney/envMRMEGAfm")
```
